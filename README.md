# ModuSynth_API

ModuSynth backend

# Create Database

- Download postgres SQL from https://www.postgresql.org/download/
- open terminal from "...\PostgreSQL\14\bin" and run psql -U postgres
- run init_modusynth_db.sql

# Start project

- Install NodeJS from https://nodejs.org/en/download/
- Clone Project
- run run_local.bat
